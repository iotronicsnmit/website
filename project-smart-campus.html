Smart Campus Network

A comprehensive IoT-enabled campus monitoring system with real-time data analytics, environmental sensing, and automated building management capabilities.

Duration: 8 months
Team Size: 6 members
Category: IoT Systems

Project Overview

The Smart Campus Network project aims to create an intelligent infrastructure that enhances campus life through interconnected IoT devices and data-driven insights. Our system monitors environmental conditions, optimizes energy consumption, and provides real-time information to students, faculty, and facility managers.

Key Objectives

  • Reduce energy consumption by 30% through intelligent automation
  • Improve campus safety with real-time monitoring systems
  • Enhance user experience through data-driven services
  • Create a scalable platform for future IoT integrations

Technical Implementation

Hardware Components

  • Arduino ESP32: Primary microcontroller for sensor data collection
  • Environmental Sensors: Temperature, humidity, air quality monitoring
  • Motion Sensors: Occupancy detection for lighting automation
  • Smart Relays: HVAC and lighting control systems
  • Gateway Devices: LoRaWAN and WiFi mesh network nodes

Software Stack

  • Backend: Node.js with Express framework
  • Database: MongoDB for time-series data storage
  • Real-time Processing: Apache Kafka for data streaming
  • Analytics: Python with TensorFlow for predictive modeling
  • Frontend: React.js dashboard with real-time visualizations

Cloud Infrastructure

  • AWS IoT Core: Device management and secure communication
  • AWS Lambda: Serverless data processing functions
  • CloudWatch: System monitoring and alerting
  • S3: Data lake for historical analytics
  • API Gateway: Secure API management
