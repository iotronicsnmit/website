SWAT

Satellite Based Water Analysis and Tracking

A comprehensive IoT solution for real-time water tanker monitoring and tracking system designed to combat water tanker mafias and ensure transparent water distribution through GPS tracking and automated water level detection.

Duration: 6 months
Team Size: 2 members
Category: IoT Systems

Problem Statement

Managing and tracking water levels in moving water tankers poses significant challenges in the logistics and utility sectors. Most current systems do not provide a reliable, real-time solution for determining whether a tanker is filled, partially filled, or empty. Additionally, GPS-based tracking of tankers is essential for operational transparency and monitoring, providing a counter measurement to water tanker mafias.

Key Objectives

  • Develop a prototype using GPS functionality for real-time tanker tracking
  • Integrate sensor that monitors water level using external non-contact sensor
  • Transmit sensor and location data over cellular network
  • Visualize data using cloud-based platform with dashboard and map widget
  • Ensure scalable design for real-world deployment

Technical Implementation

Hardware Components

  • Arduino Uno R4 Wi-Fi: Central control unit for processing
  • JSN-SR04T: Waterproof ultrasonic sensor for water level detection
  • SIM A7672S Dev Board: GSM/LTE + GNSS module for communication and GPS
  • Power Supply Module: DC power supply with breakout boards
  • IP65 Enclosure: Transparent waterproof protection

Software Stack

  • Firmware: Arduino IDE for code development
  • Communication: AT Commands for GSM/GNSS modules
  • Protocols: HTTP/HTTPS/MQTT for data transmission
  • Cloud Platform: TagoIO for device linking and visualization
  • Security: HTTPS encryption for secure data transfer

Network Infrastructure

  • 4G/LTE Network: Primary communication over cellular network
  • 2G Compatibility: Backward compatible for wider coverage
  • GNSS Module: Satellite-based GPS positioning
  • SIM Card: Standard nano SIM with data plan
  • Real-time Dashboard: Live monitoring with map widget

System Flow & Architecture

1

Data Collection

JSN-SR04T collects water level data while SIM A7672S collects GPS location from satellites

2

Processing

Arduino Uno R4 reads both data points and processes the combined information

3

Transmission

Data sent through GSM/LTE network using SIM A7672S over secure HTTP/HTTPS protocols

4

Visualization

TagoIO dashboard displays tanker location on map and current water level status

Innovation & Future Scope

Project Highlights

  • 4G/LTE Communication: Advanced cellular network instead of 2G for faster, reliable data transmission
  • Autonomous Design: Tamper-proof system preventing unauthorized access or GPS disable
  • Universal Compatibility: Suitable for both hybrid and non-hybrid water tankers
  • External Installation: Non-invasive sensor placement on tanker body

Additional Features (Future)

  • Fuel Tank Monitoring: Track vehicle fuel levels for complete logistics management
  • Tire Pressure Monitoring: Ensure vehicle safety and maintenance alerts
  • Driver Details: Store and track driver information and contact details
  • Vehicle Information: Complete vehicle registry with color, number plate data
